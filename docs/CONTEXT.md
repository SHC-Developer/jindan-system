## 도메인 모델

- **대메뉴 = 프로젝트**  
  하나의 “프로젝트”가 상단/사이드에서 선택되는 단위.

- **중메뉴 = 5개 자동화 기능**  
  한 프로젝트 안에서 물량표, 사진첩 등 5개 기능으로 구분.

- **탭 = 채팅 / 자동화**  
  - **채팅 탭**: 해당 프로젝트·기능 단위의 대화/파일.
  - **자동화 탭**: 해당 기능의 업무 자동화 UI (물량표, 사진첩 등).

구조 요약: `프로젝트(대메뉴) → 기능(중메뉴) → [채팅 탭 | 자동화 탭]`

|------|-----|------|
| 메인 | `#19647E` | 버튼, 강조, 네비게이션 |
| 서브 | `#28AFB0` | 보조 강조, 링크/호버 |

| 카드/패널 | `#FFFFFF` | 카드, 사이드바 패널 |
| 텍스트 기본 | `#1A1A2E` | 본문 |

| 경고/에러 | `#DC2626` | 에러 메시지, 경고 |
