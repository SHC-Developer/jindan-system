================================================================================
                    출퇴근 기록부 사용 안내 (직원용)
================================================================================

■ 접속 방법
  1. 배포된 웹 페이지 링크로 접속합니다.
  2. 구글(Google) 계정으로 로그인합니다.
  3. 메뉴에서 [출퇴근 기록부]를 클릭해 해당 페이지로 이동합니다.


■ 화면에 보이는 것 (서울 시간 기준)
  · 오늘 날짜와 현재 시각
  · [출근하기] 또는 [퇴근하기] 버튼
  · 오늘 근무 시간 (0h 0m ~ 실제 근무 시간)
  · 이번 주 총 근무 시간 / 지각 횟수
  · 연차 캘린더 (월별)


■ 출근하기
  · 아침에 출근했을 때 한 번 누릅니다.
  · 09:10 이전에 누르면 → 정상 출근으로 기록됩니다.
  · 09:10 이후에 누르면 → 지각으로 처리되며, 지각 사유를 입력하는 창이 뜹니다.
    (사유 입력 후 [출근하기]를 다시 눌러야 출근이 완료됩니다.)
  · 당일 연차로 등록된 날에는 출근하기 버튼이 보이지 않습니다.


■ 퇴근하기
  · 퇴근할 때 [퇴근하기] 버튼을 누릅니다.
  · 퇴근 버튼을 누르지 않아도, 18:10이 지나면 자동으로 18:10 퇴근 처리됩니다.
    (나중에 페이지를 열어도 이미 18:10으로 기록되어 있을 수 있습니다.)


■ 야근 (18:00 이후에 퇴근한 경우만)
  · 18:00 이후에 퇴근(또는 18:10 자동 퇴근)한 날에만 [야근 시작] 버튼이 보입니다.
  · 야근을 하실 때: [야근 시작] → 작업 후 [야근 종료]를 눌러 주세요.
  · [야근 종료]를 누르지 않아도, 당일 23:00이 지나면 23:00에 자동으로 야근 종료 처리됩니다.


■ 연차 사용
  · 아래쪽 [연차 캘린더]에서 사용할 날짜를 클릭해 연차로 지정합니다.
  · 여러 날을 선택한 뒤 [연차 요청 (N일)] 버튼을 누르면 관리자에게 승인 요청이 갑니다.
  · 관리자 승인 후에는 해당 날짜가 "연차(승인)"으로 표시됩니다.
  · 연차로 지정한 날에는 출근이 기록되지 않습니다.


■ 하루 흐름 요약
  1. 출근 시: [출근하기] 클릭 (09:10 넘었으면 지각 사유 입력)
  2. 퇴근 시: [퇴근하기] 클릭 (안 눌러도 18:10에 자동 퇴근)
  3. 야근 시: 18:00 이후 퇴근한 뒤 [야근 시작] → 끝나면 [야근 종료]
  4. 연차 쓸 날: 캘린더에서 날짜 선택 후 [연차 요청]으로 승인 요청


■ 문의
  사용 중 오류가 나거나 기록이 이상하면 관리자에게 문의해 주세요.
  현재 나머지 일반채팅 기능 및 프로젝트 기능은 구현 중입니다. 
  업무 지시사항은 기능구현이 완료되었으나 추후 직원들에게 적용될 예정입니다.
